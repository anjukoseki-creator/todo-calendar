# ✅ TantoList — ToDoカレンダー by 小関

家族・グループのタスクを「誰が・何を・いつ」で管理できるToDoカレンダーアプリです。
企画書「To Do カレンダー by 小関(TantoList)」をもとに、サーバー・DB不要の構成(HTML/CSS/Vanilla JavaScript + localStorage)で実装しています。

🌐 公開URL: https://anjukoseki-creator.github.io/todo-calendar/

## 主要機能

### ToDoリスト
- 追加・編集・削除(タスクをクリックで編集)
- 重要度(高・中・低)— カレンダー上で色分け表示
- 開始〜終了時間の設定
- 繰り返し設定(毎日・毎週・毎月)— 日ごとに個別に完了管理
- 日次・週次・月次フィルタ
- 期限切れタスクの強調表示とバッジ

### カレンダー
- 月間・週間・日間表示(時間軸対応、現在時刻ライン付き)
- 重要度カラー / 担当者カラードット表示

### メンバー(担当者)管理
- メンバーの追加・削除、タスクへの担当者割り当て
- メンバー別フィルタ表示
- ※サーバーレス構成のため、リアルタイム共有の代わりにJSONの書出/読込でデータを受け渡しできます

### 振り返り
- 週次・月次の達成率を自動集計(プログレスバー表示)
- 振り返りコメントの投稿・履歴閲覧

### PWA・通知
- スマホのホーム画面にアイコン追加(PWA対応、オフラインでも動作)
- 開始1時間前のブラウザ通知

## 企画書からの変更点

| 企画書 | 本実装 | 理由 |
|---|---|---|
| PHP / MySQL | HTML / CSS / JavaScript のみ | サーバー・DB不要の要件のため |
| ユーザー登録・ログイン | なし | サーバーレスのため不要 |
| 招待コードでグループ共有 | メンバー管理 + JSON書出/読込 | サーバーなしで実現できる代替手段 |
| FullCalendar.js | 自作カレンダー | 依存ライブラリゼロ化 |

## 使い方

ブラウザで `index.html` を開くか、公開URLにアクセスするだけです。
データは端末のブラウザ(localStorage)に保存され、外部には送信されません。

## ファイル構成

- `index.html` — アプリ本体(単一ファイル)
- `manifest.json` / `sw.js` / `icon-*.png` — PWA用

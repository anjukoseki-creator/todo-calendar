# ✅ TantoList — ToDoカレンダー by 小関

ToDo・カレンダー・月の目標・メモを1画面で管理できるパーソナルToDoカレンダーアプリです。
企画書「To Do カレンダー by 小関(TantoList)」をもとに、サーバー・DB不要の構成(HTML/CSS/Vanilla JavaScript + localStorage)で実装しています。

🌐 公開URL: https://anjukoseki-creator.github.io/todo-calendar/

## 主要機能

### ToDoリスト
- 追加・編集・削除(タスクをクリックで編集)
- 重要度(高・中・低)— カレンダー上で色分け表示
- 開始〜終了時間の設定
- 繰り返し設定(毎日・毎週・毎月)— 日ごとに個別に完了管理
- 日次・週次・月次フィルタ
- 期限切れタスクの強調表示とバッジ

### カレンダー
- 月間・週間・日間表示(時間軸対応、現在時刻ライン付き)
- 重要度カラー表示
- 月ごとの「🎯 目標」をタイトル下に表示・クリックで編集

### メモ(タブ式)
- 複数のメモをタブで切り替え(追加・名前変更・削除)
- 入力すると自動保存

### 目標プランナー(🧠プランタブ)
- 目標・期間(週)・週に使える時間・タイプを入力すると、目標を細分化した週ごとのスケジュールを自動生成
- 作成エンジンを選択可能:
  - **Chrome内蔵AI(Gemini Nano)** — 無料・キー不要・端末内で動作(対応ブラウザのみ。対応環境では自動で選択されます)
  - **Claude API** — 最高品質(Anthropic APIキーが必要・従量課金。キーは端末内にのみ保存)
  - **テンプレート** — AIなしの段階テンプレート生成。どの環境でも動作し、AI失敗時の自動フォールバックにもなります
- 「カレンダーに登録」で全タスクを開始日から週ごとに自動配置

### 振り返り
- 週次・月次の達成率を自動集計(プログレスバー表示)
- 振り返りコメントの投稿・履歴閲覧

### PWA・通知
- スマホのホーム画面にアイコン追加(PWA対応、オフラインでも動作)
- 開始1時間前のブラウザ通知

## 企画書からの変更点

| 企画書 | 本実装 | 理由 |
|---|---|---|
| PHP / MySQL | HTML / CSS / JavaScript のみ | サーバー・DB不要の要件のため |
| ユーザー登録・ログイン | なし | サーバーレスのため不要 |
| 招待コードでグループ共有 | なし(個人用) | サーバーレスのため。バックアップ・移行用にJSON書出/読込あり |
| FullCalendar.js | 自作カレンダー | 依存ライブラリゼロ化 |
| — | 月の目標・タブ式メモ | 追加要望による拡張 |

## 使い方

ブラウザで `index.html` を開くか、公開URLにアクセスするだけです。
データは端末のブラウザ(localStorage)に保存され、外部には送信されません。

## ファイル構成

- `index.html` — アプリ本体(単一ファイル)
- `manifest.json` / `sw.js` / `icon-*.png` — PWA用

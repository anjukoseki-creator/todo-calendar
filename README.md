# 🗓️ ToDoカレンダー

月間カレンダー上でToDoを管理できるシンプルなWebアプリです。
ビルド不要・依存ライブラリなしの単一HTMLファイルで動作します。

## 機能

- 月間カレンダー表示(前月/翌月の移動、今日へジャンプ)
- 日付を選んでToDoを追加(時刻指定も可能)
- チェックで完了、✕で削除
- 繰り返しタスク(毎日 / 毎週 / 毎月)— 日ごとに個別に完了管理
- 期限切れタスクの強調表示とバッジ
- ブラウザ通知(今日のToDo・期限切れ件数をお知らせ)
- データは localStorage に自動保存(サーバー不要)

## 使い方

`index.html` をブラウザで開くだけです。

オンライン版: GitHub Pages で公開している場合は、リポジトリの About 欄のURLからアクセスできます。

## 技術

- HTML / CSS / Vanilla JavaScript(フレームワークなし)
- 保存: `localStorage`(ブラウザごとに保存されます)
